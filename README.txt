Automatically embed a summary of a Pindula profile below a post and drastically improve the usefulness of your articles.

== Description ==

Do you want to automatically provide background information on the people and things you've written about in your article? "Pindula Mini Profiles" is focused on making reading more enjoyable for your readers by providing them contextual snippets so that they don't need to manually search the internet to understand the article's background.

== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin
2. Upload the 'pindula-mini-profiles' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and activate the plugin

= Install Via Admin Area =

1. In the admin area go to Plugins > Add New and search for "Pindula Mini Profiles"
2. Click install and then click activate

= Usage =

1. After writing a post's content, check your sidebar for suggested profiles
2. Select the profile that you want embedded at the bottom of the post
3. Alternatively type some words from the profile title in the search field and then select your desired Profile from the results(or the title itself). 
   -Your selection will be added to the bottom of the list of profiles. From there you can work with it the same way you do with the auto-suggested Profiles.

You're done!


== Frequently Asked Questions ==

= How do I get the latest Profiles? =

1. Hover over Settings and select Pindula Mini Profiles. 
2. Click on Update Profiles.   
3. The profiles will be updated for you.
